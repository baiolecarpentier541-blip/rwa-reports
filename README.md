# RWA 每日简报

由 Jarvis 自动生成，基于 CoinFound API 数据。

## 发布流程

1. 每次报告由 Jarvis 自动生成
2. 生成后 push 到本仓库
3. GitHub Pages 自动部署，访问地址：`https://<your-username>.github.io/rwa-reports/`

## 目录结构

```
rwa-reports/
├── index.html          # 最新报告（覆盖写入）
├── 2026-04-08.html     # 历史报告（按日期命名）
└── README.md
```

## 本地开发

```bash
cd ~/github/rwa-reports
# 查看当前报告
open index.html
```
